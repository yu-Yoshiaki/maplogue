## 1. デザインコンセプト

**「思考に寄り添う静かな相棒」**

- 作図ツールの「頑張って作る」印象を排し、**書くという日常動作の副産物として図が育つ**世界観を表現する
- デジタルツールでありながら「紙のノート」「アナログの思考」を連想させる温かみを持たせる
- キーワード: 手書きの温かみ / 紙の質感 / 静けさ / 自然発生 / 有機的なつながり

## 2. カラーパレット

### ブランドカラー

| トークン | 値 | 用途 |
|---|---|---|
| `--color-cream` | `#FBF4E4` | ページ背景（生成り・紙の下地） |
| `--color-cream-deep` | `#F8F0DE` | 背景のグラデーション・陰影側 |
| `--color-blue-primary` | `#1150D4` | ブランドブルー。CTA、手書き文字、接続線、選択状態 |
| `--color-ink` | `#1A1A1A` | 見出し・本文テキスト（ほぼ黒のインク色） |
| `--color-salmon` | `#F8BCA4` | 付箋・アクセントカード（サーモンピンク） |
| `--color-paper` | `#FFFFFF` | キャンバスの紙・カード面 |

### サポートカラー

| トークン | 値 | 用途 |
|---|---|---|
| `--color-tan-shadow` | `#E7D5C6` | 紙の破れ端・落ち影のベージュトーン |
| `--color-slate-muted` | `#616084` | 表の罫線・補助テキスト・UIアイコンの中間色 |
| `--color-blue-soft` | `#B8C2D6` | 接続線の淡色・非アクティブ状態 |

### 配色比率の目安

背景クリーム系 約70% / 紙・カードの白 約20% / ブルー 約7% / サーモン 約3%。
**ブルーは「書いた痕跡」にだけ使う**（CTA・手書き文字・線）。面で大きく塗らない。

## 3. タイポグラフィ

| 役割 | スタイル | 内容例 |
|---|---|---|
| アイキャッチ | 手書き風フォント、ブランドブルー、やや小さめ、見出しの直前に添える | 「書けば、図になる。」 |
| メイン見出し | 太いゴシック（明朝寄りの端正さも可）、インク色、行間広め | 「書くたびに、考えがひろがる。」 |
| サブコピー | 標準ウェイト、インク色 70〜80% 濃度、短文2行 | 「いつもの言葉を足すだけ。図は、静かに育っていきます。」 |
| カード見出し | 小さめ太字 + 絵文字/アイコン先頭 | 「📝 企画メモ」「☁ 次の一手」「⊞ 優先度」 |
| カード本文 | 小サイズ、チェックリスト・表組みは整ったUIフォント |  |

- 手書き風はアイキャッチとキャンバス上の注記のみ。**本文まで手書き風にしない**（可読性優先）
- コピーは「〜するだけ」「静かに」など受動的・穏やかな語り口で統一。命令形・煽り文句は使わない

## 4. レイアウト構成

```
┌────────────────────────────────────────────────┐
│ ◎ realtime-draw   できること 使い方 料金 ログイン [はじめる] │ ← ナビ
├────────────────────────────────────────────────┤
│                       │                        │
│  “書けば、図になる。”     │   ┌─ 破れ紙キャンバス ─┐   │
│  書くたびに、           │   │ [ツールバー]        │   │
│  考えがひろがる。        │   │  企画メモ → 次の一手  │   │
│                       │   │  利用者の声(付箋)     │   │
│  いつもの言葉を足すだけ。  │   │  優先度(テーブル)    │   │
│                       │   │  …手書き線で接続…    │   │
│  [はじめる]            │   └──────────────┘   │
└────────────────────────────────────────────────┘
```

- **2カラム構成**: 左 約35% にコピーブロック、右 約60% にプロダクト描写キャンバス
- 左カラムは上下センターよりやや上に寄せ、余白をたっぷり取る
- 右のキャンバスは背景から「浮いた紙」として配置し、ビューポート下端に少し見切れてもよい

## 5. コンポーネント仕様

### 5.1 ナビゲーション

- 左: ループ状の手書き渦巻きロゴ + ワードマーク「realtime-draw」
- 右: テキストリンク（できること / 使い方 / 料金 / ログイン）+ 塗りボタン「はじめる」
- 背景は透過でクリーム地に載せる。境界線は引かない

### 5.2 CTAボタン

- 塗り: ブランドブルー `#1150D4`、文字は白、角丸は大きめ（8〜12px 相当）
- ホバーで僅かに沈む/濃くなる程度。派手なアニメーションは世界観に合わないため避ける
- ヒーロー左下とナビ右上の2箇所に同一ラベル「はじめる」で再掲

### 5.3 キャンバス（破れ紙）

- 白い紙 `#FFFFFF` を、**ちぎったようなラフな輪郭**で背景クリームから浮かせる
- 輪郭の内側にベージュ `#E7D5C6` 系のsoft shadowを敷き、紙の厚みを出す
- 内部にドット/無地の淡い紙面。グリッド線は目立たせない

### 5.4 ツールバー

- キャンバス左端に縦配置の角丸パネル
- アイコン: ペン / 図形 / テキスト / リスト などモノクロ線画
- 選択中ツールのみブランドブルーの塗りで示す

### 5.5 カード類

| 種類 | 見た目 |
|---|---|
| メモカード（企画メモ） | 白地、細い罫線、箇条書き3行程度、角丸小 |
| リストカード（次の一手） | 白地、チェックボックス付きリスト、先頭にアイコン |
| テーブルカード（優先度） | 白地、`#616084` の細罫線、ヘッダ行あり、3〜4行 |
| 付箋（利用者の声 ほか） | サーモン `#F8BCA4`、僅かに回転（±2〜4°）、影は薄く |

- カードの影はごく薄く（紙が紙に載っている程度）。マテリアルデザイン的な強い浮きは避ける

### 5.6 接続線

- ブランドブルーの**手書き風ストローク**（微妙な揺らぎ・太さのムラ）
- 直角ポリラインではなく、有機的な曲線でカード同士を結ぶ
- 矢印は小さく控えめ。線が主役にならない太さ（1.5〜2px 相当）

## 6. 質感・エフェクト

- 背景に紙の繊維のような微細ノイズテクスチャ（不透明度 数%）を敷いてもよい
- 破れ紙エッジは SVG パス or マスク画像で表現（CSS のみの実装可否は要件側の未確定事項）
- 全体に彩度を抑え、コントラストは「インク × 紙」の関係だけで作る
- グラデーション・グロー・ガラス調（glassmorphism）は**使用禁止**。世界観が壊れる

## 7. モーション方針（実装時の参考）

- 「静かに育つ」を体現する: フェード + ごく小さな移動のみ、イージングは緩やかに
- 接続線は描画されるように（stroke-dashoffset）ゆっくり伸びる演出が世界観に合う
- バウンス・パララックス・高速スライドインは使わない

## 8. Do / Don't

| Do | Don't |
|---|---|
| クリーム地 × インク × ブルーの3色で節制する | 色数を増やす、ビビッドな差し色を足す |
| 手書き要素は「痕跡」として点在させる | 全体を手書き風フォントにする |
| 余白で静けさを作る | 要素を敷き詰めて賑やかにする |
| 影は紙の厚み程度に留める | ドロップシャドウで強く浮かせる |
| コピーは穏やかな語りで統一 | 「今すぐ」「無料！」等の煽り表現 |

## 9. アクセシビリティ留意点

- ブルー `#1150D4` × 白テキストのコントラスト比は AA を満たす（約 7:1）
- サーモン付箋 `#F8BCA4` 上のテキストはインク色 `#1A1A1A` を使う（白文字は不可）
- 手書き風フォントを使う箇所には、意味を持つテキストとして適切な代替（実テキスト/aria）を確保する

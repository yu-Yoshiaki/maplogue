# いまここハイライト・引用修正・線の根拠・ノードデザイン

作成日: 2026-07-20

## 目的

Maplogue のコア体験に近づけるため、次を追加する。

1. 直近バッチの変更箇所を「いまここ」としてハイライト（カメラ寄せなし）
2. ノードクリックで入力欄へ対象引用を差し込み、AI に部分修正させる
3. エッジの `evidence` を表示し、「なぜこの線か」を辿れるようにする
4. ノード見た目を紙×インク×ブルー基調へ調整（アイコンなし）

## 非対象

- 手動のインライン編集 UI
- カメラ自動寄せ
- AWS 等のサービスアイコン

## 主な変更

- `src/here/` … いまここ判定
- `src/quote/` … 引用フォーマット
- `SceneEdge.evidence` … 任意の根拠文字列
- `skills/scene-watch-start.md` … 引用行の解釈と evidence 付与
- `src/styles.css` … ノード／ハイライトの見た目

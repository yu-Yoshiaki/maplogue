# Maplogue LP 用操作デモ動画

## Context

LP に掲載する、Maplogue が日本語の依頼から AWS 構成マップを生成する様子を伝える無音デモ動画を作成する。メインアプリの実装には触れず、`marketing-video/` 内の自己完結した Remotion プロジェクトとして生成する。

## Assumptions

- 配信先は LP を想定し、16:9 の 1440×810 / 30 fps / 約 15 秒とする。
- 実際のバックエンド待機時間ではなく、アプリ画面に忠実なフレーム駆動の演出で生成結果を表現する。
- 音声・効果音は含めない。

## Timing

| 時間 | 内容 |
| --- | --- |
| 0.0–1.5 秒 | Maplogue の画面と入力欄を表示してフォーカス |
| 1.5–6.8 秒 | AWS 構成の依頼文をタイプライター表示 |
| 6.8–12.0 秒 | CloudFront、ALB、ECS/Fargate、RDS、S3 と矢印を順番に表示 |
| 12.0–15.0 秒 | 完成マップと控えめな結果ラベルをホールド |

## Acceptance Criteria

- [x] `marketing-video/` が独立して `npm install` / preview / render できる。
- [x] 1440×810、30 fps、約 15 秒、無音の MP4 を生成する。

- [x] CloudFront、ALB、ECS/Fargate、RDS、S3 と方向付きエッジを段階的に表示する。
- [x] 完了状態と非侵襲的な日本語ラベルを最後に表示する。
